# 프로젝트 실행 방법

```bash
# 모듈 설치
npm install

# 개발 서버 실행
npm run dev

# 테스트 실행
npm run test

# 커밋
npm run commit
```

# 커밋 규칙

- ✨ Feat: 새로운 기능 추가
- 🐛 Fix: 버그 수정
- 💄 Format: 코드 포맷 수정(ex. 세미콜론 추가 등)
- 🤖 Refactor: 코드 리팩토링(ex. UI / UX 코드 등 수정. 기능 추가, 버그 수정 x)
- ✅ Test: 테스트 코드 추가 / 수정
- 🚚 Chore: 빌드 / 라이브러리 등 환경설정 변경
- 📝 Docs: 도큐먼트 추가 / 수정
